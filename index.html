Anti Sub

Anti Sub is a desktop workflow used by creators to process subtitles and publish videos to social platforms, including TikTok integrations through authorized APIs.

Website URL for TikTok app review: https://huuhoan229.github.io/anti-sub/

What the app does

Anti Sub automates a creator workflow: video ingestion, subtitle generation, translation, TTS dubbing, rendering, and optional publishing tasks.

TikTok integration

The app uses TikTok Login Kit and Content Posting APIs. Users explicitly authorize TikTok access and manually trigger each upload operation.

Privacy controls

The app reads privacy options from TikTok creator info and follows platform constraints. Upload status is tracked and shown to the user.

How It Works

  1. User connects TikTok account through OAuth authorization.
  2. App reads creator profile and posting constraints from TikTok API.
  3. User selects rendered project and previews split and caption.
  4. User clicks Upload. App sends media through authorized TikTok endpoint.
  5. App polls publish status and shows final result to user.
